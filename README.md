# Terminal Portfolio

Static terminal-style portfolio for Fatma Buse Borlu.

## Files

```text
terminal-portfolio/
├── index.html
├── styles.css
├── app.js
└── README.md
```

## Run Locally

Open `index.html` in a browser.

For a local server:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy

This is a static site. It can be deployed to:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel
- cPanel / standard web hosting

## Custom Domain

Recommended setup:

```text
seninalanin.com       -> portfolio
projects.seninalanin.com -> future live demos
```

## Update Content

Edit `app.js`:

- `profile`
- `projects`
- command outputs

Edit `styles.css` for visual changes.
