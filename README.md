# Terminal Portfolio

Interactive terminal-style portfolio website for **Fatma Buse Borlu**, a Full Stack .NET Developer focused on ASP.NET Core, C#, SQL Server, Entity Framework Core, REST APIs, and maintainable backend architecture.

The project is designed as a lightweight personal portfolio that can be deployed quickly to GitHub Pages, Cloudflare Pages, Netlify, Vercel, or a standard hosting panel.

## Live Preview

GitHub Pages can be enabled from:

```text
Settings -> Pages -> Deploy from a branch -> main -> /root
```

Expected GitHub Pages URL:

```text
https://fatmabuseborlu.github.io/terminal-portfolio/
```

## Run Locally

Open `index.html` directly in a browser, or run a local static server:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy
